Программа предназначена для расчета оптических спектров слоистых наноструктур методом матриц переноса

Импорт данных:
Программа позволяет импортировать внешние данные (например, экспериментальные спектры отражения).
Это можно сделать через пункты меню File -> Import или File -> Import Multiple. Требования к формату данных для импорта:
 - импорт данных осуществляется из файлов формата "txt" или "dat"
 - данные должны быть представлены в табличном формате (обычно это столбец значений X и столбец значений Y; названия столбцов игнорируются)
 - допускается импорт из файла с тремя столбцами:
     -- первый столбец будет интерпретирован как X
     -- второй столбец будет интерпретирован как re(Y)
     -- третий столбец будет интерпретирован как im(Y)


Экспорт расчетных данных:
 - экспорт текущего расчета в файл (например, спектра отражения или спектра комплексного показателя преломления): File -> Export
 - множественный экспорт (в диапазоне углов падения света или в диапазоне температур): File -> Multiple Export


Работа с графиками:
 - выделение мышью области полотна с графиками позволяет изменить масштаб и зуммировать данный фрагмент
 - выделение мышью диапазона на оси X или Y позволяет изменить масштаб в рамках выделенного диапазона по соответствующей оси
 - двойной клик по области с графиками позволяет вернуться к исходному масштабу
 - нажатие с удержанием правой кнопки мыши и перемещение мыши позволяют визуально "смещать" поле с графиками
 - график можно выделить путем клика мышью по соответствующей легенде. После чего в панели Graph Manager появляется возможность редактировать имя,
   цвет, скрывать/показывать график, а также масштабировать его по осям
   (например, масштабирование по оси Y оказывается удобным для точной "подстройки" уровня нерезонансного отражения)
   NB: изменять имя и масштабировать по осям можно только импортированные кривые


Типы расчетов:
 - для структуры c количеством слоев >= 1
    - оптическое отражение (R)
    - оптическое пропускание (T)
    - оптическое поглощение (A = 1 - R - T)
 - для структуры c ровно одним слоем
    - комплексный показатель преломления (n)
    - диэлектрическая проницаемость (eps = n^2)
    - коэффициент экстинкции света по теории Ми (alphaExt)
    - коэффициент рассеяния света по теории Ми (alphaSca)


Формат описания наноструктуры:
 - конфигурация наноструктуры описывается перечислением слоев материалов, ее составляющих
 - описания слоёв группируются в блоки. Каждый блок предваряется строкой вида: "xN" (для первого блока это необязательно),
   где N >= 1 - количество повторений блока. Например, "x5" означает, что следующие несколько слоев (до следующего дескриптора повторения блока)
   будут повторяться 5 раз. Это удобно для описания периодических структур, например, последовательностей пар квантовых ям и барьеров
 - описание каждого слоя начинается с ключевого слова "type" или "material", после которого должен быть указан известный тип слоя
 - ключевое слово "material" отвечает описанию слоя из определенного материала, заранее известного (см. ниже) или определнного пользователем
 - ключевое слово "type" отвечает описанию сложного композиционного слоя (например, среда с экситонным возбуждением или системой наночастиц),
   заранее известного (см. ниже) или определнного пользователем
 - далее в произвольном порядке следует список параметров слоя (например, толщина, способ расчета показателя преломления света и т.д.)
 - описание параметров слоя можно распределить на несколько строк. Заканчивается описание слоя символом ";"

Пример описания структуры с 30 КЯ GaAs толщиной 10нм с экситонным состоянием с резонансной энергией 1.63эВ:
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
material: GaAs, eps: adachi_simple, d: 90, df: 0.0;
material: AlGaAs, eps: adachi_simple, d: 110, df: 0.0, cAl: 0.35;

x30
type: excitonic,
medium: { material: GaAs, eps: adachi_gauss, df: 0.0 },
exciton: { w0: 1.63, G0: 0.0005, G: 0.01 },
d: 10;

material: AlGaAs, eps: adachi_simple, d: 100, df: 0.0, cAl: 0.35;
// this is a single-line comment
/*
    this is a multi-line comment
*/
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Далее приведены подробные примеры описания поддерживаемых типов слоев с примерами используемого синтаксиса:

1. Слой GaAs
    Формат описания:
        material: GaAs, eps: adachi_simple, df: 0.1, d: 5;
    Параметры:
        eps - диэлектрическая проницаемость. Допустимые значения для слоев на основе GaAs:
            a) adachi_simple - простая модель Адачи [1]
            b) adachi_T - расширенная модель Адачи для четверного сплава Al(x)Ga(1-x)As(y)Sb(1-y),
               учитывающая зависимость оптических свойств компонент раствора от температуры [2]
            с) adachi_gauss - модифицированная модель Адачи из [1] (без температурной зависимости),
               в которой параметры уширения оптических переходов описываются при помощи гауссианов [3]
            d) adachi_mod_gauss - модифицированная модель из п. c), в которой мнимая компонента показателя преломления im(eps)
               при энергиях ниже края фундаментального поглощения рассчитывается по формуле re(eps) * df
        df - числовой коэффициент для вычисления im(eps) по формуле im(eps) = re(eps) * df // df означает damping factor
              Работает, когда указаны модели adachi_simple или adachi_mod_gauss, иначе - игнорируется
        d - толщина слоя [нм]
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

    Параметры:
        cAl - концентрация Al в диапазоне от 0 до 1

           material: AlGaN, d: 5, cAl: 0.3;

6. Слой материала с заданным показателем преломления
    Формат описания:
        material: custom, d: 90, eps: (см. ниже);
    Параметры:
        eps - диэлектрическая проницаемость, равная вещественному/комплексному числу или задаемая выражением
            Формат описания 1 (постоянная диэлектрическая проницаемость, заданная в форме вещественного числа):
                eps: 12.5
            Формат описания 2 (постоянная диэлектрическая проницаемость, заданная в форме комплексного числа (re(eps) = 12.5, im(eps) = 0.3)):
                eps: (12.5, 0.3)
            Формат описания 3 (диэлектрическая проницаемость, задаваемая формулой):
                eps: {
                    val v = Pi * 0.05
                    fun f(a, b) = sin(a * b) * 0.5
                    return f(x, v)
                }
            Параметры:
                val - объявление переменной
                fun - объявление функции
                return обозначает выражение, которое следует вычислить. В данном случае это вещественное число, равное значению функции f(x, v),
                где v - объявленная ранее переменная, а x - текущее значение длины волны (энергии) с оси абсцисс
            NB: детальный справочник по доступным функциям см. в отдельной вкладке "Expressions Help"

            Формат описания 4:
                eps: {
                    val v = Pi * 0.05
                    fun f(a, b) = sin(a * b) * 0.5
                    fun g(a) = cos(a * 0.1) * 0.5
                    return (f(x, v), g(x))
                }
            Параметры:
                return обозначает выражение, которое следует сосчитать (комплексное число C: re(C) = f(x, v), im(C) = g(x))
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
7. Слой материала с экситонным возбуждением
    Формат описания:
        type: excitonic,
        medium: { material: GaAs, eps: adachi_simple, df: 0.0 },
        exciton: { w0: 1.5, G0: 0.0005, G: 0.1, wb: 1.5, Gb: 0.0002, B: 1, C: 1 },
        d: 12;
    Параметры:
        medium - описание материала среды (см. п. 1 - 4)
        exciton - описание экситонного состояния
            Параметры:
                w0 - резонансная частота экситона [эВ]
                G0 и G - радиационное и нерадиационное экситонные уширения [эВ]
                // следующие параметры взяты из статьи [5] eq. 9
                wb - разность энергий дна зоны проводимости и вершины валентной зоны
                    wb = w0 + E_ex, где E_ex - энергия связи экситона (2D: 9мэВ в GaAs или 25 мэВ в GaN, 3D: 4 мэВ в GaAs)
                Gb - параметр уширения для межзонных оптических переходов в КЯ (Gb примерно равняется G)
                B - амплитудный параметр, равный f_exc / (N_cv * f_b), который определяет отношение сил осциллятора экситона
                    и переходов в непрерывном спектре. Теоретически он должен составлять от 2Ry (3D) до 16Ry (2D) экситонных ридбергов
                C - комплексный аддитивный параметр, дополняющий экситонный вклад в диэлектрическую проницаемость:
                    eps_QW = eps_medium * (1 + excitonic_contribution + C)

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
8. Слой композитного материала в модели эффективной среды
    Формат описания:
        type: eff_medium,
        medium: { material: AlGaAs, eps: adachi_simple, df: 0.0, cAl: 0.3 },
        particles: (см. ниже),
        d: 1000, f: 0.01;
    Параметры:
        f - объемная доля наночастиц
        particles - описание сферической наночастицы
            Формат описания 1:
                particles: { material: Drude, w: 14.6, G: 0.5, eps_inf: 1.0 }
            Параметры:
                type - тип наночастиц, диэлектрическая проницаемость которых вычисляется по теории Друде
                w - частота плазмонного резонанса (ПР) в вакууме [эВ]
                G - полная ширина пика ПР на половине высоты [эВ]
                eps_inf - высокочастотная диэлектрическая проницаемость

            Формат описания 2:
                particles: {
                    material: Drude_Lorentz, w: 14.6, G: 0.5, eps_inf: 1.0,
                    oscillators: {
                        1: { w: 1, f: 2, g: 3 },
                        2: { w: 4, f: 5, g: 6 }
                    }
                }
            Параметры:
                material - материал наночастиц, диэлектрическая проницаемость которых вычисляется в модели Друде-Лоренца [6]
                oscillators - описание набора лоренцевых осцилляторов в формате i: { w: value, f: value, g: value }
                    Параметры:
                        i - номер осциллятора
                        w - резонансная частота осциллятора [эВ]
                        f - "силовой" параметр
                        g - параметр уширения [эВ]

            Формат описания 3:
                particles: { material: custom, eps: (см. ниже) }
            Параметры:
                material - материал наночастиц, диэлектрическая проницаемость которых равна вещественному/комплексному числу или задается выражением
                eps - диэлектрическая проницаемость (см. выше в п.6)
            Формат описания 4:
                particles: { material: Sb }
            Параметры:
                material - материал наночастиц, диэлектрическая проницаемость которых равна диэлектрической проницаемости объемной Sb
                           из экспериментальных исследований отражения света в работе [7]

9. Слой, представляющий квадратную решетку сферических частиц, оптические свойства которой рассчитываются в модели [8]
    Формат описания:
        type: spheres_lattice,
        medium: { material: AlGaAs, eps: adachi_simple, df: 0.0, cAl: 0.3 },
        particles: { eps: Drude, w: 14.6, G: 0.5, eps_inf: 1.0 },
        d: 40, lattice_factor: 8.1;
    Параметры:
        lattice_factor - масштабирующий коэффициент, равный отношению постоянной квадратной решетки сферических частиц к их радиусу
                         (радиус частиц полагается равным d / 2)

10. Слой композитного материала для расчета дисперсии коэффициентов экстинкции и поглощения света по теории Ми.
   Допустимые режимы расчета: Extinction Coefficient, Scattering Coefficient
    Формат описания:
        type: mie,
        orders: all,
        f: 0.01,
        d: 0,
        medium: { material: AlGaAs, eps: adachi_simple, df: 0.0, cAl: 0.3 },
        particles: { eps: Drude, r: 10.6, w: 14.6, G: 0.5, eps_inf: 1.0 };
    Параметры:
        orders - учитываемые порядки разложения ряда рассеяния. Допустимые значения: 1, 2, all
        particles - описание сферической наночастицы
            Параметры:
                r - радиус частицы [нм]

Определение пользовательских материалов и типов

Данная функция предназначена для удобства переиспользования одних и тех же материалов/типов в описании структуры,
слои которых отличаются, например, только толщиной. Для этого необходимо составить дефиницию оптического материала/типа
с указанием пользовательского имени и описанием материала/типа (как при работе с обычным слоем) за исключением толщины.

NB: определения пользовательских материалов/типов должны быть приведены в самом начале описания структуры до перечисления слоев

1. Определение пользовательских материалов
    Формат описания:
        def: {
            name: custom_GaN,

            material: custom,
            eps: {
                fun f(q)=5.1529+(92842.09/(q*q-86436))
                return (f(x), 0)
            }
        };
    Параметры:
        name - имя пользовательского материала

2. Определение пользовательских типов
    Формат описания:
        def: {
            name: custom_InGaN_QW,

            type: excitonic,
            medium: { material: custom, eps: (6.7599, 0) },
            exciton: { w0: 3.18, G0: 0.00025, G: 0.04, wb: 3.205, Gb: 0.027, B: 1000000, C: (0, 0) }
        };

Пример описания структуры:

    def: {
        name: custom_GaN,

        material: custom,
        eps: {
            fun f(q)=5.1529+(92842.09/(q*q-86436))
            return (f(x), 0)
        }
    };

    def: {
        name: custom_InGaN_QW,

        type: excitonic,
        medium: { material: custom, eps: (6.7599, 0) },
        exciton: { w0: 3.18, G0: 0.00025, G: 0.04, wb: 3.205, Gb: 0.027, B: 1000000, C: (0, 0) }
    };


    material: custom_GaN, d: 87.2;

    x10
    type: custom_InGaN_QW, d: 2.2;
    material: custom_GaN, d: 74.5;
